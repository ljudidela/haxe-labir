# Fatal Labyrinth Remake (Heaps)

A roguelike game built with Haxe and Heaps.io.

## Prerequisites

1. Install Haxe 4.3+
2. Install HashLink

## Installation

Run the following commands to install dependencies:

```bash
haxelib install heaps
haxelib install hlsdl
haxelib install hlhashlink
```

## Building and Running

Compile the project:

```bash
haxe build.hxml
```

Run the game:

```bash
hl bin/main.hl
```

## Controls

- **Arrow Keys**: Move
- **Enter**: Select / Confirm
- **Escape**: Return to Menu
